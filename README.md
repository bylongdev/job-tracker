# ✅ Job Tracker

A modern, developer-friendly **job application tracking** app built with **TypeScript** (and a mostly TypeScript codebase) to help you organize your job search in one place.

> Repo: https://github.com/bylongdev/job-tracker

---

## ✨ Features

- 🗂️ Track job applications in one dashboard
- 🔎 Store key details (company, position, status, links, notes)
- 🧾 Keep your application history organized
- ⚡ Fast, clean, responsive UI
- 🧠 Simple workflow designed for everyday use

## 🚀 Getting Started

### 1) Clone the repo

```bash
git clone https://github.com/bylongdev/job-tracker.git
cd job-tracker
```

### 2) Install dependencies

Use the package manager your project is configured for:

```bash
npm install
# or
pnpm install
# or
yarn
```

### 3) Run the app locally

```bash
npm run dev
# or
pnpm dev
# or
yarn dev
```

Then open:

- http://localhost:3000
